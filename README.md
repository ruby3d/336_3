# Лабораторна робота №4

Репозиторій створено для виконання лабораторної роботи №4  
з дисципліни «Технологія створення програмних продуктів».

Студент: Костюченко Маркіян Євгенович  
Група: 336

## Структура репозиторію

- `PROGRAM` — програмний проєкт з лабораторної роботи №2;
- `HELP` — файл довідкової системи з лабораторної роботи №3.
